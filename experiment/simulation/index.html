Flow Control : Stop-and-Wait ARQ

Follow these steps:

NOTE: For better understanding we have not used the sequence number for representing frames instead we have taken the running serial number i.e., 1,2,3,4,5.
1. Select the first frame, i.e., Frame 1, to be sent to the receiver and wait to receive the acknowledgment of the 1st frame.
2. After ACK 1 is received, select the next frame to be sent.
3. ACK 2 is lost. Wait for the timeout message to appear, then retransmit Frame 2.
4. After ACK 2 is received, select the next frame to be sent.
5. After ACK 3 is received, select the next frame to be sent.
6. Frame 4 is lost. Wait for the timeout message to appear, then retransmit Frame 4.
7. After ACK 4 is received, select the next frame to be sent.
8. After ACK 5 is received, ALL FRAMES SENT!
9. Input the correct total number of frames transmitted and click the Submit button.
